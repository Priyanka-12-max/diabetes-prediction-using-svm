# diabetes-prediction-using-svm

Project Objective:
The objective of this project is to predict whether a person is diabetic or not based on medical diagnostic parameters. A Support Vector Machine (SVM) classification model is used to analyze patient data and provide accurate predictions that can assist in early detection of diabetes. 

Tech Stack:
Programming Language: Python
Libraries: NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn
IDE / Tools: VS Code, Jupyter Notebook
Version Control: Git, GitHub

Dataset
Dataset used: Pima Indians Diabetes Dataset
Features include:
Pregnancies
Glucose
BloodPressure
SkinThickness
Insulin

BMI
DiabetesPedigreeFunction
Age
Target Variable: Outcome (0 = Non-Diabetic, 1 = Diabetic)
